# Todo Task Timer (TTT)

![CI Status](https://github.com/behro/timers/actions/workflows/ci.yml/badge.svg)

A robust, interval-based task timer for focused work sessions.

## Features

- **Task Playlist**: Create a sequence of tasks with specific durations.
- **Interval Logic**: Set tasks to repeat every X laps (e.g., "Break" every 3rd lap).
- **Growth Factor**: increasing duration for repetitive tasks (e.g., +10% duration each recurrence).
- **Session Runner**: visual progress bars for Task, Lap, and Session.
- **Theme Support**: Dark/Light mode.

## Setup

1. Clone repository.
2. Run `npm install`.
3. Run `npm start` to launch the dev server.
